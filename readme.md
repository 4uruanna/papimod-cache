# DotEnv Papi Module

## Prerequisites

[Papimod/Dotenv](https://github.com/4uruanna/papimod-dotenv)

## Description

Active DI and router cache when `ENVIRONMENT` (.env) is set to `PRODUCTION`.

## Configuration

### Constants

|REQUIRED   |NAME                       |TYPE       |DEFAULT    |DESCRIPTION                                |
|:-         |:-                         |:-         |:-         |:-                                         |
|FALSE      |`PAPI_TMP_DIRECTORY`       |`string`   |`tmp`      | Relative temporary directory path         |

### Environment

|REQUIRED   |NAME                       |TYPE       |DESCRIPTION                                |
|:-         |:-                         |:-         |:-                                         |
|FALSE      |`ENVIRONMENT`              |`string`   | Project mode                              |
